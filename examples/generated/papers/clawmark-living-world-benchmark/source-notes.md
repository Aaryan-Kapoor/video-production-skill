# Source Notes: ClawMark

Selected paper: **ClawMark: A Living-World Benchmark for Multi-Turn, Multi-Day, Multimodal Coworker Agents** (arXiv:2604.23781v1)

Why this paper: submitted on 2026-04-26, it targets agent capability evaluation beyond static, single-turn tasks: multi-day workflows, changing state, multiple service backends, multimodal raw evidence, and deterministic scoring.

Primary/official sources used:
- arXiv abstract/PDF: https://arxiv.org/abs/2604.23781
- arXiv PDF: https://arxiv.org/pdf/2604.23781
- Official project site: https://claw-mark.com/
- Official GitHub repository: https://github.com/evolvent-ai/ClawMark

Citation notes:
- arXiv lists the paper as submitted 26 Apr 2026.
- arXiv abstract reports 100 tasks, 13 professional scenarios, five sandboxed services, 1537 deterministic Python checkers, seven frontier agent systems, strongest weighted score 75.8, and best strict Task Success 20.0%.
- The project site describes ClawMark as a living-world benchmark across multiple working days, services, and raw multimodal evidence with rule-based scoring, no LLM-as-judge.
